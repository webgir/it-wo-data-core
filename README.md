# IT-WO Data Core (IWDC)

**IT-WO Data Core (IWDC)** — независимая платформа обработки инженерных каталогов.
Задача системы: превратить табличные данные (XLS/XLSX) в нормализованный JSON-слой
с полной валидацией, diff/историей изменений и готовностью к интеграции в web/API.

IWDC не привязана к конкретному фреймворку или сайту.
Сайты Vitron 2.0, Wilma и любые другие системы — всего лишь клиенты
данного data-layer.

## Основные цели

- Единый канонический источник данных для всей продуктовой линейки.
- Полностью автоматический конвейер: XLS → CSV → JSON → валидация → отчёты.
- Исключение ручного редактирования JSON и контента на уровне сайта.
- Готовность к многоязычности и B2B-интеграциям (API, дилеры, мобильные приложения).
- Юридически оформляемая технология (часть IT-WO Core).

## Ключевые компоненты

- **ETL-пайплайн**: `scripts/xls-to-csv.mjs`, `scripts/csv-to-json.mjs`.
- **Схемы данных**: JSON Schema в `schemas/`.
- **Валидация**: `scripts/validate.mjs` + доменные правила.
- **Diff/Changelog**: `scripts/diff.mjs`, `scripts/changelog.mjs`.
- **CLI-обёртки**: `tools/cli/*` для удобного запуска.
- **Документация**: `docs/*` как единый «мозг» системы.
- **Поддержка Cursor**: `.cursor/*` с правилами и контекстом проекта.
